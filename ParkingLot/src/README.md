
Design Parking Lot

Requirements:
Has multiple floors
Different spot types: Car, Bike, Truck
Unpark and calculate fee
Generate ticket when parked
Allocate nearest available spot of correct type
Should be able to extend to multiple vehicle types, fee strategy etc

User Flow:

User comes in a vehicle -> Generates ticket (Contains spot, floor,  vehicle number) -> parks nearest available spot -> Unpark -> Generate fee and pay

Entities:
ParkingLot
Vehicle (Car, Bike, Truck)
Ticket 
ParkingFloor
Spot
Fee
